# 02_完成式

## 核心概念
完成式主要用來表達「在某個時間點之前，已經完成、持續或有經驗的動作」。
公式：`Have / Has / Had / Will have + p.p. (過去分詞)`

## 1. 現在完成式 (Present Perfect)
*   **公式**：`has / have + p.p.`
*   **用法**：
    1.  **經驗**：從過去到現在曾經做過的事。(常搭配 ever, never, before, once, times)
    2.  **持續**：從過去持續到現在的動作或狀態。(常搭配 for + 一段時間, since + 過去時間點)
    3.  **完成**：剛剛完成或尚未完成的動作。(常搭配 just, already, yet)
*   **多益必考關鍵字**：
    *   `lately` / `recently` (最近)
    *   `over/in/for the past/last few years` (在過去幾年來)
    *   `so far` / `up to now` (到目前為止)

## 2. 過去完成式 (Past Perfect)
*   **公式**：`had + p.p.`
*   **用法**：表達「過去的過去」。當兩個發生在過去的動作，先發生的用過去完成式，後發生的用簡單過去式。
*   **例句**：The train **had left** before I **arrived** at the station. (火車先開走，我才到)
*   **多益考點**：常搭配 `by the time + 過去式子句` (到了...的時候)。
    *   By the time he *arrived*, the meeting *had ended*.

## 3. 未來完成式 (Future Perfect)
*   **公式**：`will have + p.p.`
*   **用法**：表達「在未來的某個特定時間點之前，將會完成的動作」。
*   **多益考點**：常搭配 `by + 未來時間` 或 `by the time + 現在式子句` (代替未來)。
    *   We **will have finished** the project *by next Friday*.
    *   *By the time* you *read* this, I **will have left**.

## 📝 多益秒殺解題技巧
看到 `for + 一段時間`、`since + 過去時間`、`over the past...` ➔ 秒選 **現在完成式**。
看到 `by the time + 過去式` ➔ 秒選 **過去完成式**。
看到 `by + 未來時間` 或 `by the time + 現在式` ➔ 秒選 **未來完成式**。
