# 01_名詞子句

## 核心概念
名詞子句 (Noun Clause) 是一個完整的子句 (有主詞和動詞)，在句子中扮演「名詞」的角色，可以作為主詞、受詞或補語。多益常考引導名詞子句的連接詞選擇。

## 1. 引導名詞子句的連接詞 🌟

| 連接詞 | 用法 | 範例 |
| :--- | :--- | :--- |
| **that** | 引導確定的事實或陳述 (子句完整，that 可省略) | I believe **(that)** the project **will succeed**. |
| **whether / if** | 引導「是否」的疑問 (yes/no question) | Please let me know **whether** you **can attend**. |
| **what** | 引導「什麼」的疑問 (缺受詞或主詞) | I don't understand **what** he **meant**. |
| **who / whom** | 引導「誰」的疑問 | We don't know **who** **will be** in charge. |
| **when / where** | 引導時間/地點的疑問 | Please confirm **when** the meeting **starts**. |
| **how** | 引導「如何/多少」的疑問 | She knows **how** to *operate* the system. |
| **why** | 引導「為什麼」的疑問 | No one knows **why** the system **crashed**. |

> [!important] 名詞子句的語序
> 名詞子句必須使用**陳述語序 (S + V)**，不能使用疑問句語序！
> - ❌ I don't know **where is** he.
> - ✅ I don't know **where he is**.

## 2. 名詞子句的三種位置

### A. 作主詞
*   **That** the plan **failed** was unexpected. (名詞子句作主詞)
*   **That** customer satisfaction **has improved** is evident. (常用於正式文書)

### B. 作受詞 (最常見)
*   I know **(that)** the report **is** due tomorrow.
*   They agreed **(that)** changes **were necessary**.
*   Please confirm **whether** you **will attend** the event.

### C. 作補語
*   The problem is **(that)** we lack the budget.
*   My suggestion is **(that)** we **postpone** the meeting. ← (意志動詞規則：原形動詞)

## 3. whether vs. if 的差異

| | **whether** | **if** |
| :--- | :--- | :--- |
| 作主詞 | ✓ | ✗ |
| 介系詞後 | ✓ | ✗ |
| 後接 or not | ✓ (whether or not) | ✗ (if or not ✗) |
| 作受詞 | ✓ | ✓ |

*   **Whether** he attends **or not** doesn't matter. (✓ 作主詞，可接 or not)
*   Let me know **whether** / **if** you are available.

## 📝 多益秒殺解題技巧
1.  空格引導一個完整的子句，且在動詞後面 ➔ 先看子句是否缺主詞或受詞：
    *   **子句完整** (有 S + V) ➔ 選 `that` 或 `whether`。
    *   **子句缺主詞/受詞** ➔ 選 `what`, `who`, `which`。
2.  看到「是否」的語意 ➔ 選 `whether` (作主詞或介系詞後) 或 `if` (只作受詞)。
3.  名詞子句的動詞語序：確認是否為**陳述語序 (S + V)**，不是疑問句倒裝語序。
