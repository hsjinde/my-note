# Obsidian 插件安裝與配置指南

這裡記錄了常用的 Obsidian 插件及其相關設定方式。

## 🔄 同步相關插件

### Obsidian Git
- **用途**：利用 GitHub 進行版本控制與跨裝置同步。
- **說明**：適合熟悉 Git 操作的使用者，提供免費且完整的版本歷程記錄。

### Remotely Save
- **用途**：利用雲端硬碟（如 OneDrive、Dropbox、WebDAV 等）進行同步。
- **說明**：設定相對簡單，非常適合用來解決行動裝置（手機/平板）與電腦間的筆記同步問題。

## 🤖 AI 輔助插件

### Claudian
提供在 Obsidian 內部直接與 AI 助理互動的功能，極大地提升筆記整理與開發效率。

> [!tip] 搭配 DeepSeek API 配置範例
> 若想使用 DeepSeek 的 API 作為後端模型來節省成本，可以參考以下環境變數設定：

```env
ANTHROPIC_BASE_URL="https://api.deepseek.com/anthropic"
ANTHROPIC_AUTH_TOKEN="sk-8da0fa281c5f4fb891120afd6f3696e6"
ANTHROPIC_MODEL="deepseek-chat"
```

> [!warning] 安全提示
> 上方的 `ANTHROPIC_AUTH_TOKEN` 是敏感資訊，請妥善保管您的 API Key，不要上傳到公開的 GitHub 儲存庫。

## 📁 檔案管理插件

### Custom Attachment Location
- **用途**：自訂附件（圖片、PDF等）的儲存位置與命名規則，避免根目錄被附件塞滿，讓 Vault 保持整潔。

> [!note] 附件自動命名設定
> 建議將附件設定為根據時間戳記自動命名，可避免檔名衝突。參數配置如下：
> ```text
> file-${date:{momentJsFormat:'YYYYMMDDHHmmssSSS'}}
> ```
