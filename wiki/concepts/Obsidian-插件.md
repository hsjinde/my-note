# Obsidian 插件

這個 vault 常用的 Obsidian 插件清單與配置說明。

## 同步相關

### Git
- 用途：透過 GitHub 進行版本控制與跨裝置同步
- 適合熟悉 Git 的使用者，免費且有完整版本歷程
- 建議設定：
  - auto commit and sync
  - pull on startup

### Remotely Save
- 用途：透過雲端硬碟（OneDrive、Dropbox、WebDAV）同步
- 設定簡單，適合解決手機/平板與電腦的同步問題
- 詳見 [[Obsidian]] 實體頁

## AI 輔助

### Claudian
- 用途：在 Obsidian 內直接與 AI 助理互動
- 開啟方式：`Ctrl+P` → chat view
- 詳見 [[Claudian]] 實體頁

## 檔案管理

### Custom Attachment Location
- 用途：自訂附件（圖片、PDF）的儲存位置與命名規則
- 建議：根據時間戳記自動命名，避免檔名衝突
- 範例格式：`file-${date:{momentJsFormat:'YYYYMMDDHHmmssSSS'}}`
