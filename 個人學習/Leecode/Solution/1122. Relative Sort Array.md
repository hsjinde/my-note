- 將arr2 list 中的數字拿出來一一比對，如果arr1裡面有就pop掉，並放進res中

```python=
class Solution:
    def relativeSortArray(self, arr1: List[int], arr2: List[int]) -> List[int]:
        res = []

        while arr1 and arr2:
            
            try:
                index = arr1.index(arr2[0])            
                res.append(arr1.pop(index))
            except ValueError:
                arr2.pop(0)
        arr1.sort()
        return res + arr1
```

time complexity: $O(n * m + m log m)$, If n and m are similar in size, this can simplify to $O(n^2)$ in the worst case.
space complexity: $O(n)$
