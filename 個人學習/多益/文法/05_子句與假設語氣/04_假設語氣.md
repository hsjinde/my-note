# 04_假設語氣

## 核心概念
假設語氣用來表達與事實相反的假設、願望、或是某種條件下可能發生的事。它的特色是「**時態會往後退一格**」。

## 1. 與現在事實相反 (Present Unreal)
*   **條件句公式**：`If + S + 過去式動詞 (be動詞一律用 were), S + would/could/should/might + V (原形)`
*   **例句**：If I **were** you, I **would not accept** the offer. (如果我是你，我就不會接受。 -> 但我不是你)

## 2. 與過去事實相反 (Past Unreal)
*   **條件句公式**：`If + S + had + p.p. (過去完成式), S + would/could/should/might + have + p.p.`
*   **例句**：If he **had studied** harder, he **would have passed** the exam. (如果他當時有努力讀書，他當時就會及格了。 -> 但他當時沒讀書)

## 3. 與未來事實相反 / 未來極小可能性 (Future Unreal)
*   **條件句公式**：`If + S + should + V, S + 助動詞 + V / 祈使句` (萬一...)
*   **例句**：If you **should** have any questions, please **contact** me. (萬一你有任何問題，請聯絡我。)

## 📝 多益必考重點 (秒殺技巧)

### 考點 1：條件句的倒裝 (If 省略)
多益極愛考 If 省略的倒裝句型，請將助動詞移到主詞前面。
1.  **與過去相反倒裝**：`Had + S + p.p., S + would have p.p.`
    *   *Had* I *known* the truth, I would have told you. (原句: If I had known...)
2.  **與未來相反倒裝** (最常考)：`Should + S + V, 祈使句`
    *   *Should* you *need* further assistance, please let us know. (原句: If you should need...)
3.  **與現在相反倒裝** (較少考)：`Were + S ..., S + would V`
    *   *Were* I the manager, I would fire him.

### 考點 2：Without / But for (要不是因為...)
*   與現在相反：`Without / But for + N, S + would + V`
*   與過去相反：`Without / But for + N, S + would have + p.p.`
