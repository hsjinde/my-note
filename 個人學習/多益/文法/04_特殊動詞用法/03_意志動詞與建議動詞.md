# 03_意志動詞與建議動詞

## 核心概念
意志動詞 (Verbs of Volition) 與建議動詞 (Verbs of Suggestion) 後面接的 that 子句中，動詞必須用**原形動詞 (should + V 或直接省略 should，稱為「假設現在式」/「虛擬語氣現在式」)**。這是多益必考的重要文法點！

## 1. 常見的意志動詞與建議動詞 🌟

這些動詞後面的 `that 子句`，動詞要用**原形動詞**：

| 類別 | 動詞 |
| :--- | :--- |
| **意志/命令** | insist, demand, require, order, command |
| **建議/請求** | suggest, recommend, propose, request, urge, ask |
| **決定** | decide (有時適用) |

## 2. 句型結構 🌟

**公式**：`S + 意志/建議動詞 + that + S + (should) + 原形動詞`

注意：`should` **可以省略**，但原形動詞必須保留，即使主詞是第三人稱單數也**不加 -s**。

| 動詞 | 正確範例 | 說明 |
| :--- | :--- | :--- |
| **suggest** | The manager suggested that he **submit** the report. | 不是 submits 或 submitted |
| **recommend** | I recommend that she **attend** the training. | 不是 attends |
| **require** | The policy requires that all employees **wear** ID badges. | 不是 wears |
| **insist** | He insisted that the meeting **be** held immediately. | be，不是 is |
| **demand** | The client demanded that we **revise** the proposal. | 不是 revises |
| **request** | She requested that the deadline **be** extended. | be，被動也用原形 |

## 3. 名詞形式也適用
相同的動詞若變成名詞，之後的 that 子句同樣使用原形動詞。
*   *His* **suggestion** *that the meeting be postponed* was accepted.
*   *The* **requirement** *that all applicants submit a portfolio* is strict.

## 4. 形容詞形式 (It is + adj. + that ...)
以下形容詞後面的 that 子句，同樣需要原形動詞：
*   `important, necessary, essential, vital, crucial, advisable, imperative, urgent`
    *   It is **essential** that everyone **be** present at the meeting.
    *   It is **important** that she **submit** the form before the deadline.

## 📝 多益秒殺解題技巧
1.  看到 `suggest, recommend, require, insist, demand, request` 後面有 that 子句 ➔ that 子句的動詞秒選**原形動詞**，不管主詞是誰。
2.  看到 `It is important/necessary/essential that...` ➔ 子句動詞秒選**原形動詞**。
3.  **陷阱**：`suggest` 和 `recommend` 還可以接 `V-ing` (不接 that 子句時)。
    *   I suggest **reviewing** the report. (V-ing，沒有 that 子句) ✓
    *   I suggest that she **review** the report. (that 子句，原形動詞) ✓
