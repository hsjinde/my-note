# ==方法 一  Unordered Map==
利用Unordered Map，利用每個字串的組合作為key，而value就是索引值
- Sorting Strings: 確保了字母排列將具有相同的排序
- Grouping Anagrams: 我們檢查排序後的字符串是否存在於Map中。如果存在，我們將當前字符串添加到答案向量中的相應組中。如果不存在，我們則創建一個新的組並將其索引存儲在Map中。

```python!
class Solution:
    def groupAnagrams(self, strs: List[str]) -> List[List[str]]:
        mp = {}  # 字典，用來儲存排序後字串與其在結果中的索引
        res = []  # 最終的結果列表
        
        for s in strs:  # 遍歷輸入的字串列表
            # 將字串排序後作為鍵
            sorted_s = ''.join(sorted(s))
            
            # 如果這個鍵不在字典中，表示這是一組新的異位詞
            if sorted_s not in mp:
                # 將這個新鍵對應到結果列表的新索引
                mp[sorted_s] = len(res)
                # 並新增一個包含該字串的子列表到結果中
                res.append([s])
            else:
                # 如果鍵已經存在於字典中，直接將字串加入對應的列表
                res[mp[sorted_s]].append(s)
        
        return res  # 返回結果列表

```

#Time complexity: O(n∗k∗logk) (K is the length of the longest string)
#Space complexity:  O(n∗k∗logk) (K is the length of the longest string)
