# 06_五大句型

## 核心概念
英文所有句子都可以歸納為五大基本句型。理解句型結構有助於判斷空格的詞性，也是閱讀測驗分析長句的基礎。

**縮寫說明**：S = 主詞、V = 動詞、O = 受詞、C = 補語、IO = 間接受詞、DO = 直接受詞

## 1. 五大句型總覽 🌟

| 句型 | 結構 | 範例 |
| :--- | :--- | :--- |
| **句型一** | S + Vi (不及物動詞) | The market **grows** rapidly. |
| **句型二** | S + V (連綴) + SC (主詞補語) | The manager **seems** *confident*. |
| **句型三** | S + Vt + O | She **submitted** *the report*. |
| **句型四** | S + Vt + IO + DO | He **gave** *me* *a raise*. |
| **句型五** | S + Vt + O + OC (受詞補語) | They **elected** *him* *president*. |

## 2. 各句型詳解

### 句型一：S + Vi
動詞本身語意完整，不需受詞。常見不及物動詞：appear, arrive, occur, rise。
*   The accident **occurred** at noon.

### 句型二：S + V(連綴) + SC
連綴動詞後面接補語說明主詞的狀態，補語必須是**名詞或形容詞**，不能是副詞！
*   常見連綴動詞：be, seem, appear, become, remain, stay, look, sound
*   The new plan sounds **reasonable**. (reasonable 是形容詞，修飾主詞 plan)

### 句型三：S + Vt + O
最常見的句型。及物動詞後方必須有受詞。
*   The company **launched** *a new product* last week.

### 句型四：S + Vt + IO + DO
「授與動詞」(give, send, offer, show, tell) 可接兩個受詞：先接「人 (間接受詞)」，再接「物 (直接受詞)」。
*   可改寫為：S + Vt + DO + **to/for** + IO
    *   He **gave** *her* *a present*. = He **gave** *a present* **to** *her*.

### 句型五：S + Vt + O + OC
受詞補語 (OC) 說明的是「受詞」的狀態，OC 可以是名詞或形容詞。
*   常見動詞：make, find, consider, keep, call, elect, appoint, name
    *   They **appointed** *her* *director*. (director 說明受詞 her)
    *   We **found** *the meeting* *useful*. (useful 說明受詞 the meeting)

## 📝 多益秒殺解題技巧
1.  **分辨句型二與句型三**：句型二的動詞是連綴動詞 (remain, become)，後面接的是主詞**補語 (形容詞/名詞)**；句型三的動詞是及物動詞，後面接的是受詞。
2.  **句型五陷阱**：`make + O + adj.` → 空格填形容詞。(e.g., The news *made* us **happy**.)
3.  **找主要動詞**：分析長句時，先找出主詞 (S)，再找主要動詞 (V)，剩下的就是受詞或補語。
