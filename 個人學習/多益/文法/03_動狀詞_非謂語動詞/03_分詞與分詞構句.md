# 03_分詞與分詞構句

## 核心概念
分詞 (Participle) 分為「現在分詞 (V-ing)」與「過去分詞 (p.p.)」，兩者都可以作形容詞修飾名詞。分詞構句 (Participial Phrase) 則是將副詞子句簡化的語法，多益閱讀題中大量出現。

## 1. 分詞作形容詞：修飾名詞 🌟

分詞可以放在名詞**前面**或**後面**修飾：

*   **現在分詞 (V-ing)**：表示「主動、進行中」的狀態，說明「正在做...的」名詞。
    *   the **attending** *employees* (正在出席的員工，員工主動出席)
    *   a **growing** *company* (正在成長的公司)
*   **過去分詞 (p.p.)**：表示「被動、完成」的狀態，說明「被...的」名詞。
    *   the **revised** *schedule* (被修改過的時間表)
    *   documents **submitted** *by the team* (由團隊提交的文件，後置修飾)

## 2. 分詞的位置

*   **前置修飾 (名詞前)**：單一分詞常放在名詞前。
    *   a **signed** contract / the **missing** file
*   **後置修飾 (名詞後)**：分詞片語 (有其他字搭配) 通常放在名詞後。
    *   the employees **working on** the new project
    *   the report **submitted** last week

## 3. 分詞構句 (Participial Phrase) 🌟
將副詞子句的「連接詞 + 主詞 (與主句相同)」省略，動詞改成分詞。

| 原句 (副詞子句) | 分詞構句簡化 |
| :--- | :--- |
| **Because** she *finished* the report, she went home early. | ***Finishing*** the report, she went home early. |
| **When** he *arrived* at the office, he found a memo. | ***Arriving*** at the office, he found a memo. |
| **After** it *was completed*, the project was reviewed. | ***Completed***, the project was reviewed. |

*   **主動 ➔ V-ing**：主詞自己「做」這個動作 ➔ 現在分詞 (Doing...)
*   **被動 ➔ p.p. (Being + p.p. 或直接省略 Being)** ➔ 過去分詞 (Done...)

## 4. 多益常考分詞慣用語

| 分詞慣用語 | 中文意思 |
| :--- | :--- |
| **Regarding / Concerning** | 關於... |
| **Considering** | 考慮到... |
| **Given (that)** | 鑒於 / 假設... |
| **Compared to/with** | 與...相比 |
| **Based on** | 基於... |
| **Judging from** | 從...判斷 |
| **Speaking of** | 說到... |

*   **Regarding** your inquiry, please find the attached file.
*   **Given** the current market conditions, we should be cautious.

## 📝 多益秒殺解題技巧
1.  **修飾名詞 (形容詞位置)**，判斷分詞：名詞「主動做」 ➔ **V-ing**；名詞「被做/已完成」 ➔ **p.p.**。
2.  **分詞構句開頭**，判斷省略的主詞是「主動做」還是「被動做」來選 V-ing 或 p.p.。
3.  看到 `Regarding`, `Given`, `Considering` 等 ➔ 這是「獨立分詞構句」，視為固定表達直接使用。
4.  **陷阱**：分詞構句的隱含主詞必須與主句主詞一致，否則是「懸垂分詞」(dangling participle)，多益閱讀理解可藉此判斷邏輯。
