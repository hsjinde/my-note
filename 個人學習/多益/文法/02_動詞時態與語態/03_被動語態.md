# 03_被動語態

## 核心概念 (被動的概念)
當句子的主詞是「動作的接受者」時，就必須使用被動語態。
在英文中，被動語態的標準公式是：**`be 動詞 + p.p. (過去分詞)`**

## 1. 為什麼要用被動語態？
*   **不知道或不在乎動作是誰做的**：The report was completed yesterday. (誰完成的不重要，重點是報告完成了)
*   **強調接受動作的對象**：The new policy was announced by the CEO. (強調「新政策」)

## 2. 各時態的被動語態變化 🌟
被動語態的時態變化，重點在於改變 **be 動詞** 的型態，後面的 p.p. 永遠不動。

| 時態 | 主動語態 | 被動語態公式 (be + p.p.) | 被動範例 |
| :--- | :--- | :--- | :--- |
| **現在簡單** | write / writes | `am/is/are + p.p.` | is written |
| **過去簡單** | wrote | `was/were + p.p.` | was written |
| **未來簡單** | will write | `will be + p.p.` | will be written |
| **現在進行** | is writing | `am/is/are + being + p.p.` | is **being** written (常考) |
| **現在完成** | has written | `have/has + been + p.p.` | has **been** written (常考) |
| **助動詞** | can write | `can + be + p.p.` | can be written |

## 3. 多益必考重點：主動與被動的判斷 (秒殺技巧) 🌟
多益非常喜歡在選項中同時放「主動」和「被動」的動詞，讓你判斷。
**判斷法則：看動詞後方有沒有「受詞 (名詞)」！**

*   絕大多數的及物動詞 (transitive verbs)：
    *   **主動**：動詞後方 **有** 受詞 (名詞)。
        *   The company **will hire** *new employees*. (主動：公司將雇用新員工)
    *   **被動**：動詞後方 **沒有** 受詞 (通常接介系詞如 by, in, for，或者直接句號)。
        *   *New employees* **will be hired** (by the company). (被動：新員工將被雇用)

## 4. 例外：授與動詞 (給予類動詞)
像 give, offer, send, award 等動詞有「兩個受詞」(人 + 物)。
變成被動語態時，如果把「人」當主詞，後面**還會剩下一個受詞(物)**。
*   主動：They **offered** *him* *a job*.
*   被動：He **was offered** *a job*. (注意：這裡是被動，但後面依然有受詞 a job)

## 📝 多益秒殺解題技巧
1.  **先找主詞與動詞的關係**：主詞是「做」這個動作，還是「被做」？物通常是被做 (The email was sent)。
2.  **看空格後方**：如果空格後是介系詞 (by, in, to) 或句號，極高機率選 **被動語態 (be + p.p.)**。
