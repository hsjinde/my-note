# 03_形容詞與副詞

## 核心概念
形容詞與副詞是多益詞性題的核心。**形容詞**修飾名詞；**副詞**修飾動詞、形容詞或其他副詞。辨別兩者的關鍵在於「它修飾的對象是什麼」。

## 1. 必背形容詞字尾 (字尾判斷法) 🌟

| 字尾 | 例字 |
| :--- | :--- |
| **-ful** | successful, helpful, useful |
| **-less** | careless, hopeless, effortless |
| **-ous / -ious** | numerous, serious, previous |
| **-al / -ial** | annual, financial, additional |
| **-ive / -ative** | effective, productive, creative |
| **-able / -ible** | available, responsible, flexible |
| **-ent / -ant** | efficient, significant, relevant |
| **-ing / -ed** | (情緒動詞) interesting / interested |

## 2. 必背副詞字尾

*   最常見：**形容詞 + -ly** ➔ quickly, efficiently, carefully
*   注意：不是所有 -ly 結尾的字都是副詞！(friendly, costly, timely 是**形容詞**)
*   常見不加 -ly 的副詞：`hard`, `fast`, `well`, `late`, `early`, `high`
    *   He worked **hard**. (副詞) / a **hard** worker. (形容詞)
    *   ⚠️ `hardly` (幾乎不) ≠ `hard` (努力地)；`lately` (最近) ≠ `late` (晚)

## 3. 形容詞 vs. 副詞的位置 (多益考點) 🌟

| 詞性 | 修飾對象 | 常見位置 |
| :--- | :--- | :--- |
| **形容詞** | 名詞 | **名詞前**、連綴動詞 (be/seem/remain) **後** |
| **副詞** | 動詞、形容詞、副詞 | 動詞前後、形容詞/副詞**前** |

*   **形容詞位置**：
    *   a **successful** *launch* (名詞前)
    *   The plan seems **effective**. (連綴動詞後，補語)
*   **副詞位置**：
    *   The project was completed **efficiently**. (修飾動詞)
    *   She did an **extremely** *good* job. (修飾形容詞)

## 4. 多益常混淆組合

| 形容詞 | 副詞 | 辨別要點 |
| :--- | :--- | :--- |
| high (高的) | highly (非常、高度地) | *highly* qualified (非常有資格的) |
| near (附近的) | nearly (幾乎) | *nearly* finished (幾乎完成) |
| hard (努力的、困難的) | hardly (幾乎不) | *hard*ly *any* rain (幾乎沒有下雨) |
| late (晚的) | lately (最近) | I've been busy *lately*. |

## 📝 多益秒殺解題技巧
1.  **空格前有冠詞 (a/the) 或後有名詞** ➔ 秒選**形容詞**。
2.  **空格後修飾動詞 (如 work, complete, run)** ➔ 秒選**副詞** (-ly)。
3.  **空格修飾空格後的形容詞** (如 ___ effective) ➔ 秒選**副詞**。
4.  **看到連綴動詞 (seem, remain, become) 後面的空格** ➔ 秒選**形容詞**，不選副詞。
