# 01_不定詞

## 核心概念
不定詞 (Infinitive) 的形式為 **`to + 原形動詞`**，在句子中可作名詞、形容詞或副詞使用。多益常考「接不定詞還是動名詞」的辨別，以及「不定詞作目的/結果的副詞用法」。

## 1. 不定詞的三種功能

### A. 作名詞 (主詞 / 受詞 / 補語)
*   **作主詞**：To learn English takes time. (但較常用動名詞作主詞)
*   **作受詞**：I decided **to postpone** the meeting.
*   **作補語**：Our goal is **to improve** customer satisfaction.

### B. 作形容詞 (修飾名詞)
放在名詞後方，說明名詞的用途或內容。
*   He is the first employee **to arrive** every day.
*   We need a plan **to address** these issues.

### C. 作副詞 (表目的 / 結果)
*   **表目的** (為了)：She went to the store **to buy** supplies.
*   **in order to / so as to**：更正式的「為了」，多益書信文體常用。
    *   We updated the system **in order to** *improve* performance.

## 2. 必須接不定詞的動詞 🌟
看到這些動詞，後面接的動詞一定是 **to V**：
*   **意志/計畫類**：want, wish, hope, plan, intend, decide, choose
*   **嘗試/承諾類**：agree, promise, offer, refuse, manage
*   **其他**：expect, need, fail, tend, afford

| 動詞 | 範例 |
| :--- | :--- |
| **want** | I want **to attend** the seminar. |
| **plan** | We plan **to expand** into new markets. |
| **manage** | She managed **to finish** the project on time. |
| **fail** | He failed **to submit** the report. |
| **afford** | We cannot afford **to lose** this client. |

## 3. 不定詞 vs. 動名詞：語意改變的動詞 🌟

| 動詞 | to V (未來/刻意) | V-ing (過去/一般) |
| :--- | :--- | :--- |
| **remember** | remember *to do* (記得去做) | remember *doing* (記得曾做過) |
| **forget** | forget *to do* (忘了去做) | forget *doing* (忘了曾做過) |
| **stop** | stop *to do* (停下來去做) | stop *doing* (停止做...) |
| **try** | try *to do* (試圖做) | try *doing* (試試看做) |

*   Please **remember to** *submit* the form by Friday. (記得去交)
*   I **remember** *meeting* her at the conference. (記得曾見過她)

## 4. 不定詞的省略 (bare infinitive)
在使役動詞 (make, have, let) 和感官動詞 (see, hear, watch) 後，不定詞的 **to** 要去掉，直接用**原形動詞**。
*   Let me *know* your decision. (不是 to know)
*   I saw him *leave* the office. (不是 to leave)

## 📝 多益秒殺解題技巧
1.  看到 `want, plan, decide, manage, fail` 後面的空格 ➔ 秒選 **to + V**。
2.  看到 `in order to` 後面的空格 ➔ 秒選**原形動詞 V**。
3.  看到 `remember / forget + ___`：要判斷語意是「未來的行動」(to V) 還是「過去的記憶」(V-ing)。
4.  看到 `make / let / have / see / hear + 受詞 + ___` ➔ 秒選**原形動詞 V** (省略 to)。
