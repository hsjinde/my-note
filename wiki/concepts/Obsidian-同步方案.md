# Obsidian 同步方案

Obsidian 筆記的同步策略比較，涵蓋免費與付費方案。

## 本 Vault 現行方案

**Google Drive 串流**：vault 建在 Google Drive 資料夾內，自動同步。

## 其他免費方案

| 方案 | 優點 | 缺點 |
|---|---|---|
| Git + GitHub | 版本控制完整 | 需技術背景 |
| Remotely Save 插件 | 支援多種雲端（S3/WebDAV/Dropbox） | 設定較複雜 |
| Syncthing | P2P 同步、無雲端依賴 | 需多台裝置在線 |

## Obsidian 官方同步（付費）

- 端到端加密
- 支援版本歷史
- 月費 ~$8 USD

## 手機端使用

- iOS/Android 皆有 Obsidian App
- 需搭配同步方案才能讀取 vault
- Git 方案在手機端較不方便

## 導出選項

- 純 Markdown 本身即可移植
- 可用 Pandoc 轉 PDF/DOCX
- 知識圖譜可截圖分享
