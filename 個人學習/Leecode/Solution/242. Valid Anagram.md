# 方法一: Sorting
```python!
class Solution:
    def isAnagram(self, s: str, t: str) -> bool:
        return sorted(s) == sorted(t)
```
由於排序操作，此方法的時間複雜度為 O(n log n)，其中 n 是字符串的長度。

# 方法二:Hash Table

```python!
#from collections import defaultdict
class Solution:
    def isAnagram(self, s: str, t: str) -> bool:
        count = defaultdict(int)
        for i in s:
            count[i] += 1
        for i in t:
            count[i] -= 1
        for val in count.values():
            if val != 0: 
                return False
        return True
```
這個解決方案的時間複雜度是 O(n)，其中 n 是兩個字符串中字符的總數。
