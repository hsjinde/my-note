# 排序演算法

將資料按特定順序排列。分為**比較排序**（O(n log n) 下界）與**非比較排序**（可達 O(n)）。

## 複雜度對照表

| 演算法 | 最佳 | 平均 | 最差 | 空間 | 穩定 |
|---|---|---|---|---|---|
| Bubble Sort | O(n) | O(n²) | O(n²) | O(1) | ✓ |
| Selection Sort | O(n²) | O(n²) | O(n²) | O(1) | ✗ |
| Insertion Sort | O(n) | O(n²) | O(n²) | O(1) | ✓ |
| Shell Sort | — | O(n^1.3) | O(n²) | O(1) | ✗ |
| Merge Sort | O(n log n) | O(n log n) | O(n log n) | O(n) | ✓ |
| Quick Sort | O(n log n) | O(n log n) | O(n²) | O(log n) | ✗ |
| Heap Sort | O(n log n) | O(n log n) | O(n log n) | O(1) | ✗ |
| Bucket Sort | O(n+k) | O(n+k) | O(n²) | O(n+k) | ✓ |
| Radix Sort | O(nk) | O(nk) | O(nk) | O(n+k) | ✓ |

## 比較排序

### Bubble Sort（氣泡排序）
逐一比較相鄰元素，錯誤順序則交換。每回合將最大值推到末尾。

### Selection Sort（選擇排序）
從未排序區找最小值，放到已排序區末端。

### Insertion Sort（插入排序）
逐一取元素插入已排序區的正確位置。對近乎排序的資料效率佳。

### Shell Sort（希爾排序）
插入排序改良版。引入 Gap 間距分組，逐回合縮小 Gap，最後 Gap=1。

### Merge Sort（合併排序）
**分治法**：分割至單元素 → 兩兩合併時排序 → 最終合併。穩定 O(n log n)。

### Quick Sort（快速排序）
選 Pivot → 小於放左、大於放右 → 遞迴兩邊。平均最快，最差 O(n²)（可用三數中位數選 Pivot 改善）。

### Heap Sort（堆積排序）
建 Max Heap → Root 與末尾交換 → 移除末尾 → Heapify → 重複。O(n log n) 且 O(1) 空間。

## 非比較排序

### Bucket Sort（桶排序）
分配到固定區間的桶子 → 各桶內排序 → 收集。空間換時間。

### Radix Sort（基數排序）
按位數（個十百千）分配到 0-9 桶子，逐位排序。LSD（從右）或 MSD（從左）。

## 選擇指南

| 情境 | 建議 |
|---|---|
| 資料量小 | Insertion Sort |
| 近乎排序 | Insertion Sort |
| 一般用途 | Quick Sort / Merge Sort |
| 要求穩定 | Merge Sort |
| 記憶體有限 | Heap Sort |
| 整數且範圍小 | Bucket / Radix Sort |
