# Claude Code 懶人包

由 [[mathruffian-dot]] 製作的系列教學，每支影片附一個 MD 檔（懶人包），丟給 Claude Code 桌面版即可自動完成安裝與設定。

## 使用方式

1. **AI 自動安裝**：貼上 repo URL，AI 讀取 `SKILL.md` 後列出所有懶人包供選擇
2. **手動下載 MD 檔**：看影片 → 下載 MD → 丟給 Claude Code 桌面版

最低條件：Claude Pro 以上 + Claude Code 桌面版 + 網路

## 懶人包清單

| 編號 | 名稱 | 說明 |
|---|---|---|
| 00 | 環境建置 | Git、GitHub CLI、uv 等基礎工具 |
| 01 | 連接 NotebookLM | NotebookLM MCP + 簡報與圖表 |
| 02 | 連接 GitHub | GitHub + GitHub Pages 教材上線 |
| 03 | 建立第二大腦 Obsidian | Obsidian + MCP + Google Drive 同步 |
| 03+ | 第二大腦設定指南 | 三層結構 + CLAUDE.md + 模板 + 每週重整 |
| 04 | 連接 Supabase | 雲端資料庫 |
| 04.5 | 連接 Firebase | Firestore MCP CRUD（對老師更友善） |
| 05 | 安裝本地 AI Ollama | 免費、隱私、離線可用 |
| 06 | 設定 Gemini 免費 API | 不用信用卡 |
| 07 | 班級工具工作模式 | 一鍵啟動老師建專案模式 |
| 08 | ChatGPT Image 2.0 | 全域 `draw` Skill，在任何專案生圖 |

## 影片系列（16 集）

分為三類：
- **概念**：EP01 Claude 全生態、EP02 Skills
- **安裝集**：EP03/05/07/09/09.5/11/14
- **應用高潮**：EP04/06/08/10/12/13/15
- **進階**：EP16 跨電腦工作術（GDrive + Obsidian + GitHub 三方同步）
