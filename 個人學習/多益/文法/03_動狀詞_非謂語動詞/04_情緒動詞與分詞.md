# 04_情緒動詞與分詞

## 核心概念
情緒動詞本身的意思是「使...感到(某種情緒)」。例如：`interest` (使...感興趣)、`surprise` (使...驚訝)。
這些動詞轉變為分詞時，會有兩種形式：**現在分詞 (V-ing)** 和 **過去分詞 (p.p.)**，兩者的修飾對象完全不同！

## 1. 情緒分詞的變化法則
*   **V-ing (令人...)**：表示事物本身具有某種特性，通常用來修飾「**事物**」。
*   **p.p. (感到...)**：表示某人因為外在刺激而產生某種情緒，通常用來修飾「**人**」。

## 2. 常見的情緒動詞
*   **interest**：interesting (令人感興趣的) / interested (感到感興趣的)
*   **surprise**：surprising (令人驚訝的) / surprised (感到驚訝的)
*   **satisfy**：satisfying (令人滿意的) / satisfied (感到滿意的)
*   **disappoint**：disappointing (令人失望的) / disappointed (感到失望的)
*   **confuse**：confusing (令人困惑的) / confused (感到困惑的)
*   **excite**：exciting (令人興奮的) / excited (感到興奮的)
*   **exhaust**：exhausting (令人疲憊的) / exhausted (感到疲憊的)

## 3. 多益常見句型
### 句型 A：修飾事物用 V-ing
*   The presentation was very **interesting**. (簡報令人感興趣)
*   We received **disappointing** sales results. (我們收到令人失望的銷售結果)

### 句型 B：修飾人 (感受) 用 p.p.
*   I am **interested** *in* learning new skills. (我對學習新技能感到興趣)
*   The customers were **satisfied** *with* the service. (顧客對服務感到滿意)
    *   *注意：情緒 p.p. 後方常搭配特定的介系詞 (in, with, at 等)*

## 📝 多益秒殺解題技巧
遇到情緒動詞的選項 (如 tire, tiring, tired)：
1.  **找修飾對象**：看這個空格是在修飾「人」還是「事物」。
2.  **事物 ➔ 選 V-ing**。(The result is surprising.)
3.  **人 ➔ 選 p.p.**。(The manager is surprised.)
4.  **例外陷阱**：雖然說「人 p.p. / 物 V-ing」是個好記的口訣，但如果是在形容「這個人本身令人感到...」，也可以用 V-ing (He is an interesting person. 他是個有趣的人)。不過多益多半考基本型態。
