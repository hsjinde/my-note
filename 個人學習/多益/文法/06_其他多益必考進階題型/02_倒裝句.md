# 02_倒裝句

## 核心概念
倒裝句 (Inversion) 是將「助動詞或 be 動詞」移到「主詞」前面，改變一般語序的句型。多益考的倒裝句主要分為兩大類：**否定副詞倒裝**與**條件句 If 省略倒裝**（假設語氣）。

## 1. 否定副詞倒裝 🌟

當句子以**否定或半否定的副詞/副詞片語**開頭時，主詞與助動詞的位置必須互換 (倒裝)。

**公式**：`否定副詞 + 助動詞/be + 主詞 + 動詞`

| 否定/限制副詞 | 意思 | 倒裝範例 |
| :--- | :--- | :--- |
| **Never** | 從不 | **Never** *have* I *seen* such dedication. |
| **Rarely / Seldom** | 很少 | **Rarely** *does* the CEO *attend* such meetings. |
| **Hardly (ever)** | 幾乎不 | **Hardly** *had* she *left* when it started raining. |
| **Not only...but also** | 不只...還... | **Not only** *did* they *finish* on time, but they also exceeded the goal. |
| **Not until** | 直到...才... | **Not until** the report is submitted *will* the payment be released. |
| **Under no circumstances** | 在任何情況下都不 | **Under no circumstances** *should* you *share* your password. |
| **On no account** | 絕對不可以 | **On no account** *are* employees *allowed* to... |
| **Only + 副詞片語** | 只有... | **Only then** *did* they *realize* the problem. |

## 2. 條件句倒裝 (If 省略) 🌟
請參考 [[04_假設語氣]] 的考點 1。

當假設語氣中 `If` 被省略時，助動詞移到句首，形成倒裝：

| 原句 | 倒裝句 |
| :--- | :--- |
| If **had** p.p. (過去) | **Had** + S + p.p., S + would have p.p. |
| If **should** V (未來) | **Should** + S + V, ... |
| If **were** (現在) | **Were** + S ..., S + would V |

*   *Should* you **need** any assistance, please contact us. (最常考)
*   *Had* I **known** earlier, I would have acted differently.

## 3. Only + 副詞片語 開頭的倒裝

當 `Only` 加上副詞片語放在句首時，主要子句必須倒裝：
*   **Only after** the meeting **did** she **realize** the issue.
*   **Only by** working together **can** we **achieve** this goal.
*   **Only when** you **submit** the form **will** the process **begin**.

> [!note]
> 注意：`Only + 主詞` 開頭時**不倒裝**！
> - **Only he** *knew* the truth. (只有他知道，主詞是 he，不倒裝)

## 4. So/Neither/Nor 的附和倒裝

*   **So + 助動詞 + 主詞**：表示「...也是如此」(肯定附和)
    *   A: I enjoyed the conference. B: **So did** I. (我也是)
*   **Neither/Nor + 助動詞 + 主詞**：表示「...也不」(否定附和)
    *   A: I don't like long meetings. B: **Neither do** I. (我也不喜歡)

## 📝 多益秒殺解題技巧
1.  看到 `Never, Rarely, Seldom, Not only, Only + 副詞片語` 開頭 ➔ 確認後面是否有**倒裝** (助動詞在主詞前)。
2.  看到 `Should + 主詞 + V`, `Had + 主詞 + p.p.`, `Were + 主詞` 的句型 ➔ 這是**假設語氣倒裝**，後面主句記得填對時態 (would/would have)。
3.  **閱讀測驗**：倒裝句型判斷助動詞位置，是分析長句的好工具。
