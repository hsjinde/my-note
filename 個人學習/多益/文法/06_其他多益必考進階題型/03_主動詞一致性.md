# 03_主動詞一致性

## 核心概念
主動詞一致性 (Subject-Verb Agreement) 指的是動詞的單複數必須與主詞保持一致。多益常考「找出真正的主詞」，因為長句中容易被插入語或修飾語干擾，導致誤判主詞的單複數。

## 1. 基本規則

*   **單數主詞** ➔ 動詞加 **-s / -es** (第三人稱單數現在式)
*   **複數主詞** ➔ 動詞**不加 -s**

## 2. 多益陷阱：找出真正的主詞 🌟

### A. 中間修飾語的干擾
介系詞片語 (of/with/in...) 或關係子句插入主詞和動詞之間，**不影響主詞的單複數**。

*   The report *on sales figures* **is** ready. (主詞是 report，不是 figures)
*   The employees *who work in the R&D department* **are** highly qualified.
*   Each of the managers **is** responsible for a different region. (主詞是 Each，單數)

### B. 集合名詞 (Collection Nouns)

| 用法 | 說明 | 範例 |
| :--- | :--- | :--- |
| 視為單數整體 | 美式英語常見用法 | The **committee** **has** made a decision. |
| 指個別成員 | 英式英語較常見 | The **team are** divided on this issue. |

多益通常以**單數**處理集合名詞：`company, team, committee, staff, board, jury, family, audience`

### C. 不可數名詞 (永遠單數)
`information, advice, equipment, furniture, progress, research, news, evidence`
*   The **information** **is** confidential.
*   **News** of the merger **was** announced this morning.

## 3. 特殊主詞的一致性 🌟

| 主詞形式 | 一致性 | 範例 |
| :--- | :--- | :--- |
| **each / every + N** | 單數 | **Each** employee **is** required to sign. |
| **either / neither** (單獨) | 單數 | **Neither** of the proposals **was** accepted. |
| **A or B / Either A or B / Neither A nor B** | 動詞配合**靠近的主詞 (B)** | Neither the manager nor the employees **were** informed. |
| **not only A but also B** | 動詞配合**靠近的主詞 (B)** | Not only the staff but also the CEO **is** attending. |
| **A and B** | 複數 (兩人/物) | The CEO and the CFO **are** both present. |
| **There is/are** | 配合後方的**真正主詞** | **There are** several issues to discuss. |
| **The number of + N** | 單數 ("數量"本身) | **The number of** applicants **has** increased. |
| **A number of + N** | 複數 ("許多") | **A number of** applicants **have** applied. |

## 4. 分數與百分比
分數或百分比 + of + 名詞：動詞配合 **of 後方的名詞**。
*   **Fifty percent** of the workforce **is** working remotely.
*   **Three-fourths** of the documents **have** been reviewed.

## 📝 多益秒殺解題技巧
1.  **找真正的主詞**：先刪去所有 `of...`, `with...`, `in...` 等介系詞片語，剩下的才是真正的主詞。
2.  **each / every / one of** ➔ 動詞一律**單數**。
3.  **The number of** ➔ 單數；**A number of** ➔ 複數。
4.  **A or B / Neither A nor B** ➔ 動詞配合**最靠近的主詞 (B)**。
5.  **不可數名詞**（news, information, advice...）➔ 動詞一律**單數**。
