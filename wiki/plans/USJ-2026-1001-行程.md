# USJ 2026/10/01（四）整天行程規劃

## 當日資訊

| 項目 | 內容 |
|------|------|
| 日期 | 2026/10/01（四）平日 |
| 營運時間 | 預估 9:00~20:00（Halloween 可能延長） |
| 季節活動 | Halloween Horror Nights（18:00 後）、25 週年 Discover U!!! |
| 交通 | JR 大阪駅→Universal City 駅：JR 夢咲線 15 分、200¥ |
| 注意 | **已停止現場售票，必須提前線上購買門票 + Express Pass** |

## 快速通關券推薦

### 🥇 首選：Express Pass 4 Minecart & Thrills（6,800~18,800¥）

| 包含設施 | 價值 |
|---------|------|
| 🎮 瑪利歐賽車：庫巴的挑戰書 | SNW 必玩，排隊常 90~150 分 |
| 🛤️ 咚奇剛的瘋狂礦車 | 2026 最新設施，排隊最長 |
| 🦖 飛天翼龍 | 園區最刺激雲霄飛車 |
| 🚀 Space Fantasy / JAWS（二選一） | 額外省 30~60 分 |

> **含 SNW 保證入場時段**，不需搶整理券。
> 預算有限下的最佳選擇，省 3~4 小時排隊。

### 🥈 進階：Express Pass 7 Minecart & Selection（10,800~23,800¥）

多涵蓋：哈利波特禁忌之旅、小小兵瘋狂乘車遊、Hollywood Dream 等
適合第一次去、想全制霸的人。

### 🥉 頂規：Express Pass Premium（價格最高）

13 項設施無時段限制隨到隨玩，但 10 月價格可能很可觀。

> **週四平日人潮中等，Express Pass 4 就夠用**，搭配 Single Rider 可玩到所有主要設施。

## 當天完整行程

### 行前準備
- [ ] 下載 USJ 官方 App，入場前綁定門票
- [ ] 購買 Express Pass（建議數週前，熱門日售罄快）
- [ ] 查看當日 Cool Japan 抽選票（若有）
- [ ] 帶水、防曬（9 月大阪炎熱潮濕）
- [ ] 穿舒適鞋子（園區超大）

### 行程表

| 時間 | 活動 | 備註 |
|------|------|------|
| **07:30** | 抵達 Universal City 駅 | 開園前 60~90 分鐘到門口排隊 |
| **08:00~08:45** | 門口排隊 + App 確認 SNW 入場時段 | Express Pass 已含時段則免擔心 |
| **09:00~11:00** | 🎮 **Super Nintendo World** 🏰 | Mine-Cart Madness → Mario Kart → Yoshi → Power-Up Band 互動 |
| **11:00~11:30** | 🦖 飛天翼龍 | Express Pass 或 Single Rider |
| **11:30~12:00** | 🎢 Hollywood Dream – The Ride | Single Rider 快速 |
| **12:00~13:00** | 🍽️ 午餐 | 建議 11:30~12:00 錯峰吃，避開 12~13 尖峰 |
| **13:00~14:00** | ⚡ **哈利波特園區** 🧙 | 禁忌之旅 → Flight of the Hippogriff → 活米村 |
| **14:00~15:00** | 🟡 **Minion Park** 💣 | Despicable Me + Villain-Con Minion Blast（2025 新設施） |
| **15:00~16:00** | 🦈 JAWS → Space Fantasy → 侏羅紀公園乘船遊 | Single Rider 省時 |
| **16:00~16:30** | 自由時間 / 拍照 / 購物 | 或補玩遺漏設施 |
| **17:00~17:30** | 🎉 **NO LIMIT! Parade 25 週年版** | 必看遊行 |
| **18:00~20:00** | 👻 **Halloween Horror Nights** 🎃 | 18:00 後殭屍出沒，氣氛超棒 |
| **20:00** | 閉園 → JR 回大阪駅 | 或 Captain Line 渡輪往海遊館（10 分、900¥） |

### Single Rider 省時清單
- ✅ 飛天翼龍
- ✅ Hollywood Dream
- ✅ 哈利波特禁忌之旅
- ✅ JAWS
- ✅ 瑪利歐賽車（有時開放）
- ✅ 侏羅紀公園乘船遊
- ✅ Space Fantasy

> 一天搭配 Express Pass + Single Rider 可省 **5~7 小時**排隊

## 注意事項

- 🕐 **餐廳尖峰 12:00~13:00** 排隊 60~90 分鐘，建議 11:30 或 13:30 用餐
- 👻 Halloween 晚間 18:00 後殭屍出沒，膽小者可提早離開
- ☀️ 10 月大阪仍偏暖，帶水與防曬
- 🚆 回程 JR 夢咲線末班約 23:00，時間充裕
