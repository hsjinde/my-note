# Monotonic stack
```
class Solution:
    def sumSubarrayMins(self, arr: List[int]) -> int:
        MOD = 10**9 +7
        monotonic_stack = [] # (index, num)
        res = 0
        arr = arr + [float('-inf')]

        for i, n in enumerate(arr):
            while monotonic_stack and n < monotonic_stack[-1][1]:
                j, m = monotonic_stack.pop()
                left =  j - monotonic_stack[-1][0] if monotonic_stack else j + 1
                right = i - j  
                res = res + (m * left * right) 
            monotonic_stack.append((i,n))
        return res % MOD
```

### tips

1. 初始化變量 ：
    - `MOD` 是一個常數，用於防止結果過大。
    - `stack` 用於存儲數組元素及其索引，保持單調遞增性。
    - `res` 用於累加結果。
    - 將數組 `arr` 增加一個負無窮大的元素，以處理最後一個子數組。

2. 遍歷數組 ：
    - 使用 `enumerate` 遍歷數組 `arr`，得到每個元素的索引和值。
    - 如果堆疊不為空且當前元素小於堆疊頂部元素，則從堆疊中彈出頂部元素。

3. 計算子數組最小值之和 ：
    - 計算當前子數組的左邊界 `left` 和右邊界 `right`。
    - 使用公式 `(m * left * right) % MOD` 計算當前子數組的最小值對結果的貢獻，並將其累加到 `res` 中。

4. 將當前元素推入堆疊 ：
    - 將當前元素及其索引推入堆疊，保持單調遞增性。

5. 返回結果 ：
    - 返回累積的結果 `res`。



### 時間複雜度和空間複雜度
時間複雜度 ：$O(n)$。每個元素最多被推入和彈出堆疊一次。
空間複雜度 ：$O(n)$。堆疊的最大長度為 n。
